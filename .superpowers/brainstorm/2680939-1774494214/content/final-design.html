Line-by-Line Spec

L1 — Brand Identity

━━ prefix + model badge (tier-specific glyph)
Model name (gradient animated)
━━━ bridge (flex, fills remaining)
Smart CWD (parent/dir) — OSC 8 link to repo
CLI version — OSC 8 link to changelog
Account email (color by account) — OSC 8 link to console
━━ suffix

L2 — Usage + Session

5h usage: marker + bar + % + reset timer
7d usage: marker + bar + % + reset timer
━━━ bridge (flex)
Token speed: ~Nt/s (output_tokens / api_duration)
Session cost: $X.XX (SGD, blonde)
Session uptime: Xm / Xhr Ym

L3 — Context + Tokens

Context: marker + bar + % + token ratio
Tokens in: c⧈ Nk (cached) + r□ Nk (raw)
// divider
Tokens out: c⧈ Nk (cached) + w□ Nk (written)
━━━ bridge (flex) — splits for handoff warning
Inbox depth: ◇ N

L4 — Git State

Branch: ◈ name — OSC 8 link to GitHub tree
Unsynced: ↑N (blonde)
Worktree: wt:name (when active, muted)
Git changes: +N (emerald) -M (ruby)
━━━ bridge (flex)
Issues: ◈ N — OSC 8 link to issues page

Progressive Compact Mode

60% context: Hide reset timers (3h 59m) (4d 18h) on L2
75% context: Collapse token breakdown to totals: in 130k // out 274k
85% context: Hide bridges entirely, pack segments tight

Structural Features

Consistent bridges: ━━ on all 4 lines (no thin/thick mixing)
Auto-alignment: Replace first_w hack with general column alignment across L2/L3
Flex layout: Bridges distribute remaining terminal width proportionally
OSC 8 links: Branch, issues, CWD, version, email — all clickable where supported
Link kill switch: CHRYSAKI_NO_LINKS=1 env var

Cut from V2 (separate issues)

Powerline mode — alternative renderer with colored backgrounds + arrow separators (may do)
Skills tracker — hook + JSONL storage + renderer widget (future)
