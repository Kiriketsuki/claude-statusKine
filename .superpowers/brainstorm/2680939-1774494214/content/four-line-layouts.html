Comparison

A: New Metrics Line

+ Least disruption to existing layout
+ Line 4 is purely additive
- Line 1 still feels sparse
- New line is a grab bag of unrelated metrics

B: Brand Splits

+ Line 1 becomes a dramatic gradient banner
+ Workspace line groups dir + branch + session naturally
- Session cost/speed/clock mixed with git feels odd
- Brand takes 2 lines (50% of space) for little info

C: Dedicated Git Line

+ Clean separation: brand / git / usage / context
+ Git info has room for worktree, changes, links
+ Cost + clock on Line 1 = session identity
- Usage line (L3) right side feels empty
