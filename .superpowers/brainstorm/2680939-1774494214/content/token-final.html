Field Mapping

in (teal)

c⧈ = cache_read_input_tokens
Tokens reused from previous turn's cache

r□ = input_tokens
Fresh input tokens (not from cache)

out (blonde)

c⧈ = cache_creation_input_tokens
Tokens newly written to cache for reuse

w□ = output_tokens
Model-generated output tokens
