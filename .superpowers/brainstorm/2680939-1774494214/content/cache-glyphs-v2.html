Same base shape, two variants. C = cache read (retrieved), W = cache write (created). No emojis — all single-width Unicode geometric glyphs.

Filled = retrieved (full, complete), Outline = written (being formed). Same shape family, two states.

Directional Pairs

Same glyph, opposite direction. Read = inward/down (pulling from store), Write = outward/up (depositing).
