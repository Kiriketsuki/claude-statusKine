How It Works: Random Alternation

Bridge Colour Selection (per render)
# Each render, hash the timestamp to pick colours:
_seed=$(date +%s)
_c1=$(( _seed % 9 )) # L1 accent
_c2=$(( (_seed / 3) % 9 )) # L2 bridges
_c3=$(( (_seed / 7) % 9 )) # L3 bridges
_c4=$(( (_seed / 11) % 9 )) # L4 bridges + branch
# Different primes ensure each line gets a different colour,
# and the colours shift on every render (even at 1fps).
# Adjacent lines are guaranteed different at any given moment.
